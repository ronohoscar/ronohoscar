Relative CSS units At the core of responsive web design are relative CSS units. These are units that get their value from some other external value. This is handy because it allows, for example, the width of an image to be based on the width of the browser. The most common ones are: % em rem vw vh In this article, we'll start with the percentage unit %, and then we'll look at the rem unit in the final section.
